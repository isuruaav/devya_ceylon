# DEVYA CEYLON Website Image Generation Guide

## Visual direction
Create premium, realistic, warm and trustworthy Ayurveda healthcare imagery rooted in Kandy, Sri Lanka. Use natural daylight, herbal green, warm cream, muted gold and earthy brown. Keep compositions clean and website-friendly. Avoid visible brand logos, text, watermarks, exaggerated medical claims and overly dark spa imagery.

## Export settings
- Preferred format: WebP
- Quality: 80–88%
- Keep each image below approximately 350 KB when possible
- Use lowercase filenames with hyphens
- Add meaningful `alt` text in the PHP page
